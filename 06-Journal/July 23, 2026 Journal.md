> **Dual Approval means one person requests access to a privileged account, and another person must approve that request before access is granted.**

> An **Approval Request** is a request created by a user asking for permission to access a protected account or perform a privileged action.

> **Check-in = returning the account back to CyberArk.**

> **Automated Release by PSM** is a **safety feature that automatically ends a privileged account session when the user closes the PSM connection or leaves without properly logging out**.

> **Bulk** means doing the same action on **many accounts at the same time** instead of doing it one account at a time.

> **Scope** means which objects, accounts, users, or systems does this rule, permission, or setting apply to?

> **Organizational Unit (OU)** is a **container (folder) inside Active Directory that is used to organize users, computers, groups, and other objects**.

> **Discovery is a CyberArk feature that finds privileged accounts, machines, and credentials in your environment that are not yet managed by CyberArk.**

> **Onboarding** means:
> **Adding a privileged account into CyberArk so CyberArk can start protecting, managing, and controlling it.**
