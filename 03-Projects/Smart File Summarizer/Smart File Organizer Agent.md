#projects

**Tools:** Python, LangChain, LangGraph, Ollama
### Description
An agent that automatically organizes messy folders on your computer. This is an interesting project for my AI projects.
### What it should do
- Scan files in a folder
- Classify files (images, school, code, etc.)
- Rename and move them into folders
### How it works
- Python handles file system
- LangChain + Ollama decides file category
- LangGraph making [[Ideas for tools|tools]] + nodes
- Agent loops through files
### Use case
- Clean Downloads folder automatically
- Organize school or project files
