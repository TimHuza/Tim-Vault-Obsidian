#concepts 
 
- **MCP Host:** The AI application you are using (e.g., an AI-powered IDE or chat interface).

- **MCP Client:** The mechanism within the host that translates your AI's requests and receives data.

- **MCP Server:** A lightweight, standardized server built for a specific external service (like GitHub or a local database). It takes real-world information and translates it into a format the AI can understand.
